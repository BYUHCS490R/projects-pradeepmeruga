CSS Selectors

The ways to select an item in CSS are tag, ID, class, child, and universal selectors.

An ID is for one element, but a class can be used for many elements.

A span is for small text and stays on the same line, but a div is for bigger sections and starts on a new line.

A div is just a container, but header, main, and footer are special parts of a webpage.

Colors can be chosen by color names, RGB, HEX, or HSL.
