Engagement - Tables

  1. We use a table to show data in rows and columns, like schedules or grades.
  2. We do not use tables to design the page layout. We use CSS like Flexbox or Grid for that.
  3. We do extra work when the table has headers or complex information, so screen readers can understand it.
