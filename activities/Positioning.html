Engagement - Positioning

  1. Normal flow is the default way a webpage is arranged. Elements go from top to bottom. Block elements go on new lines, and inline elements stay in the same line.
  2. Float moves an element to the left or right side. Text can wrap around it.We use clear to stop things from wrapping and to keep the layout neat.
  3. Relative moves an element a little from its normal position, but it still keeps its space on the page. Absolute places an element exactly where you want it and removes it from the normal layout. Fixed keeps an element in the same place on the screen even when you scroll up or down.
  4. Make buttons bigger, add space between them, use simple colors, and use a menu icon for small screens.
