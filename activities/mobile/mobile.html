Engagement - Page for Print

  1. The viewport meta tag helps a webpage fit properly on different screen sizes. It makes the content adjust so the site looks good and is easy to read on phones, tablets, and computers.
  2. You can adjust CSS for printing by using @media print in the CSS. This allows you to change how the page looks when someone prints it, like hiding menus or changing the layout.
  3. You would adjust CSS for printing when you want the printed page to look simple and clear. For example, you might remove extra elements like navigation bars and keep only the important text and information.

Engagement - Responsive Pages

  1. The viewport meta tag helps the web page adjust its size and layout to fit different screen sizes, especially on phones and tablets.
  2. You can adjust CSS for printing by using @media print in the CSS. This allows you to change how the page looks when someone prints it, like hiding menus or changing the layout.
  3. we can use media tag in CSS to apply different styles for smaller screens like phones.
